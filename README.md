
# COVID Deaths Project Analysis

## Project Overview
This project analyzes global COVID-19 deaths and vaccination data across countries and continents. It includes total deaths, rolling vaccination counts, and population-level percentages.

## Data Files
- covid_deaths.csv — COVID deaths by country
- covid_vaccinations.csv — daily vaccination data
- covid_Pop_vs_Vac.csv — combined dataset with rolling vaccinations

## SQL Scripts
- Covid_portfilo_Project_Ferid.sql — Oracle SQL code to create views for analysis

## Objectives
- Analyze deaths by country and continent
- Calculate population percentage vaccinated
